Goal List

Each goal has an emoji (optional — for display only), a label (the name), and keywords (what the parser looks for in messages). The keywords are what actually match — so even if someone skips the emoji, it still works.

How keyword matching works: The parser searches each line of a report for the keywords. For example, if keywords are wake,5am,5 am, then a line containing "Wake up at 5AM" or "5 am: Yes" will both match. This means your group doesn't need to use exact emojis. Keywords are NOT case-sensitive.
